metricsd / debug sql

Read-only queries against the live vm_metrics DuckDB. Single statement only. DuckDB classifies the query and only SELECT / EXPLAIN are accepted — this covers WITH / FROM / VALUES / DESCRIBE / SHOW and read-only PRAGMAs. Mutating pragmas (e.g. PRAGMA memory_limit), CTE-wrapped writes, and EXPLAIN ANALYZE are rejected. Try DESCRIBE vm_metrics or SHOW TABLES.
