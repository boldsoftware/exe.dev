Hello from your new exe.dev VM

This page is served by the welcome service that ships with every new default exe.dev box.

This VM has a persistent disk, sudo, HTTPS, and SSH all wired up.

What next?

Run your own app

Stop this server and start your own.

  • ssh {{.Hostname}}@exe.dev sudo systemctl stop welcome
  • Have your server listen on port 8000

Edit this server

Use this as a starting point.

  • Source code is in ~/src
  • Edit via SSH or open in VS Code
  • Chat with our coding agent, Shelley
  • sudo systemctl restart welcome to pick up changes
