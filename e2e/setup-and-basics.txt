# Setup and Basic Operations Test

## Instructions

Test basic exed registration and fundamental commands.

The e1e framework has already started exed, exelet, and sshpiper.
Read env-info.txt for the service ports.

### Test Setup

1. Create yourself an ssh key with ssh-keygen if you don't have one.

### Test Steps

#### Registration

1. Connect to exed via sshpiper using the SSH port from env-info.txt.

2. Complete email verification using the browser.

3. Test the "whoami" command.

4. Test the "ls" command (should have no boxes yet).

## Hints

### Registration Hints
- Use: `ssh -o StrictHostKeyChecking=no -o UserKnownHostsFile=/dev/null -p <SSH_PORT> localhost`
- After entering email, get verification URL. Open in browser and click "Confirm Email Verification".
- After email verification, type `whoami` at exe.dev prompt. Shows email and SSH key fingerprint.
- Type `ls` at exe.dev prompt. Should show "No boxes found. Create one with 'new'."
