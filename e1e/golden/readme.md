These files are the on-screen results of one of the e1e test flows.

These are committed as snapshots of the state of the UX at any given commit.

If your changes modify one of these text files, please manually review the diff to make sure it is a change we want our users to experience.

If the change looks like the result of non-deterministic output, please fix the output to be deterministic.

Otherwise, if the change looks good, commit it.
