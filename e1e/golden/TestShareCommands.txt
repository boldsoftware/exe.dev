Command: share

Share HTTPS VM access with others

Usage: share <subcommand> <vm> [args...]

Options:

  show            - Show current shares for a VM
  port            - Set the HTTP proxy port for a VM
  set-public      - Make the HTTP proxy publicly accessible
  set-private     - Restrict the HTTP proxy to authenticated users
  add             - Share VM with a user via email
  remove          - Revoke a user's access to a VM
  add-link        - Create a shareable link for a VM
  remove-link     - Revoke a shareable link
  receive-email   - Enable or disable inbound email for a VM
  ssh             - Control team SSH access to a VM
