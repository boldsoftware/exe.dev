Signup Controls

/debug > signup-controls

Block New Account Creation

Warning: When enabled, users with unrecognized email addresses cannot create new accounts.

When enabled, users trying to login with an email we haven't seen before will be blocked. Existing users can still log in and add new SSH keys.

IP Abuse Filter

Info: The IP abuse filter uses IPQS to check if an IP address has been flagged for recent abuse during signup. When enabled, users signing up from flagged IPs will be blocked.

Proof-of-Work Challenge

Info: When enabled, new users must complete a proof-of-work challenge before creating an account. Difficulty is currently set to {{.POWDifficulty}} leading zero bits (~{{.POWAvgHashes}} hashes average).

Rate Limiter

Rate limit: 5 requests per minute per IP address.
