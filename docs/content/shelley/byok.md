Shelley uses exe.dev's built-in [LLM Gateway](/docs/shelley/llm-gateway) by
default, but you can use your own API keys instead. When creating a new conversation,
bring up the models menu by choosing "Add / Remove Models..." in the models drop-down.

The workflow for common providers like Anthropic, OpenAI, and Gemini is
straight-forward. If you have a z.ai coding subscription (as opposed to the
per-token endpoint), use the coding endpoint
`https://api.z.ai/api/coding/paas/v4`.
