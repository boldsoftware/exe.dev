GitHub publishes the public half of all authorized SSH keys.

If you `ssh exe.dev` with an SSH key that you've associated with your GitHub account, and it passes a few extra checks on our end, we let you speed run auth.

For more details about how this works, check out [https://words.filippo.io/whoami-updated/](https://words.filippo.io/whoami-updated/)
