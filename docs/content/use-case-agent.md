---

When you create a box with `ssh exe.dev new`, both `claude` and `codex` are pre-installed,
as well as the Shelley agent at [https://<box-name>.exe.dev:9999/](https://box-name.exe.dev:9999/).

Use the agent to do research, build prototypes, install other software, and so
on.
