# exelet network: NAT

exelet NAT networking provides local networking using a bridge. The following options
are supported:

- `bridge`: This configures the bridge name (default: `br-exe`)
- `network`: This configures the local network (default: `192.168.40.0/24`)
- `dns`: This configures the DNS nameservers to use (default: `1.1.1.1`)
- `ntp`: This configures the NTP server to use (default: `ntp.ubuntu.com`)
- `router`: This configures the local node IP to be used as a router
