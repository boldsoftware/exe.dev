Shelley

exe.dev's open-source, mobile-friendly, web-based, multi-modal, multi-model, single-user coding agent.

Shelley is a coding agent. It operates the shell with ease. We named it as a nod to the Unix shell and, yes, poet Percy Bysshe Shelley.

  • -Open-source because it runs on your VM
  • -Mobile-friendly because ideas can come any time
  • -Web-based because terminal-based scrollback is punishment for shoplifting in some countries
  • -Multi-modal because screenshots, charts, and graphs are necessary, not to mention delightful
  • -Multi-model to benefit from all the innovation
  • -Single-user so the agent runs right next to your systems
