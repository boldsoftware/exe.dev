Resource Rate
Disk Usage $0.08 / GiB / month
CPU $0.05 / core / hour
Active Memory $0.016 / GiB / hour
Inactive Memory $0.08 / GiB / month
Per-VM Minimum $0.001 / hour

CPU and active memory are billed by peak usage within each hour. Disk usage and inactive memory are billed monthly. The per-VM minimum is billed by the peak number of VMs in a given hour.

Example

A 2-CPU, 8 GiB memory VM with 20 GiB of storage running continuously for 30 days at full capacity would cost $166.48.

But a typical idle machine on our platform uses about 200 MiB of active memory, 300 MiB of inactive memory, and 0.1% of a CPU. With 20 GiB of storage, its monthly charge would be roughly $4.63.
